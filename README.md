# MAM Themes

A collection of themes for the **MyAnonamouse (MAM)** website.

---

## 🚀 Installation

1. Go to **MAM → Preferences → Custom CSS**
2. Open one of the theme files from this repo
3. Copy the entire CSS
4. Paste into the Custom CSS box
5. Save and refresh
